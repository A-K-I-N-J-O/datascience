datascience.tables.Table.__init__¶

Table.__init__(columns=None, labels=None, formatter=<datascience.formats.Formatter object>)[source]¶

Create a table from a list of column values.

>>> letters = ['a', 'b', 'c', 'z']
>>> counts = [9, 3, 3, 1]
>>> points = [1, 2, 2, 10]
>>> t = Table([letters, counts, points], ['letter', 'count', 'points'])
>>> t
letter | count | points
a      | 9     | 1
b      | 3     | 2
c      | 3     | 2
z      | 1     | 10

For other ways to initialize a table, see Table.empty(), Table.from_rows(), Table.from_records(), Table.read_table(), and Table.from_columns_dict().

Args:
columns (list): A list in which each element is another list
containing the values for a column in the order the columns appear in labels.

labels (list): A list of column labels.

formatter (Formatter): An instance of Formatter that
formats the columns’ values.
Returns:
A new instance of Table.
Raises:
AssertionError:
  • labels is specified but columns is not and vice-versa.
  • The length of labels and the length of columns are unequal.
